ESTADO joga_bot ( ESTADO  e)

Função que faz com que os bots joguem ou passem. Para isso a função está dividida em 3 partes. Uma para quando o bot joga, sendo ele o último jogador, logo joga a carta mais baixa que tiver na mão,outra para quando combinações de menos de 5 cartas e, por último, uma para as combinações de 5 cartas.
